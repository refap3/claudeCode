# Novita Games

Two pygame games generated by running [Ollama](https://ollama.com) on [novita.ai](https://novita.ai) with an NVIDIA RTX 5090 GPU using the `qooba/qwen3-coder-30b-a3b-instruct:q3_k_m` model.

## Games

### Car Racing (`car.py`)

Top-down car racing game. Dodge incoming traffic, survive as long as possible, and beat your high score.

**Controls:**
- `UP` / `DOWN` — Accelerate / brake
- `LEFT` / `RIGHT` — Steer
- `R` — Restart after game over

**Features:**
- 3 lives with a bonus life every 10 points
- Speed increases as your score grows
- Colorful obstacle cars in random lanes

**Run:**
```bash
cd novita
pip install pygame
python car.py
```

---

### Spaceship Battle (`battle.py`)

Vertical space shooter. Survive waves of descending enemy ships while managing health and lives.

**Controls:**
- `ARROW KEYS` — Move ship
- `SPACE` — Shoot (hold for continuous fire)
- `R` — Restart after game over

**Features:**
- OOP architecture: `Player`, `Enemy`, `Bullet`, `Explosion`, `Game` classes
- Procedurally generated sound effects (shoot and explosion) via `pygame.mixer`
- Enemy ships fire back with randomised cooldowns
- Health bar per ship; player has 3 lives before game over

**Run:**
```bash
cd novita
pip install pygame
python battle.py
```

## Requirements

Both games require Python 3 and pygame:

```bash
pip install pygame
```
