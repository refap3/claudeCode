# Novita Games

Two pygame games generated by running [Ollama](https://ollama.com) on [novita.ai](https://novita.ai) with an NVIDIA RTX 5090 GPU using the `qooba/qwen3-coder-30b-a3b-instruct:q3_k_m` model.

## Games

### Car Racing (`car.py` / `car.html`)

Top-down car racing game. Dodge incoming traffic, collect nitro, survive the Mad Orange President.

**Controls:**
- `UP` / `DOWN` — Accelerate / brake
- `LEFT` / `RIGHT` — Steer
- `N` (hold) — Burn nitro; auto-chains to the next pack while N is held
- `R` — Restart after game over

**Features:**
- 3 lives; nitro lets you destroy obstacles for bonus distance instead of taking damage
- Nitro packs stack — collect multiple depots and burn through them by holding N
- Oil spots randomly teleport you across the road
- Speed and traffic density increase every 10 km
- **Mad Orange President** — a super-wide oncoming car announced ~0.8 s before arrival; crushes all obstacles and kills you instantly if you don't dodge to the road edge

`car.html` is a pixel-faithful browser port (Canvas 2D API, no dependencies — just open in any browser).

**Run:**
```bash
cd novita
pip install pygame
python car.py
```

---

### Spaceship Battle (`battle.py`)

Vertical space shooter. Survive waves of descending enemy ships while managing health and lives.

**Controls:**
- `ARROW KEYS` — Move ship
- `SPACE` — Shoot (hold for continuous fire)
- `R` — Restart after game over

**Features:**
- OOP architecture: `Player`, `Enemy`, `Bullet`, `Explosion`, `Game` classes
- Procedurally generated sound effects (shoot and explosion) via `pygame.mixer`
- Enemy ships fire back with randomised cooldowns
- Health bar per ship; player has 3 lives before game over

**Run:**
```bash
cd novita
pip install pygame
python battle.py
```

## Requirements

Both games require Python 3 and pygame:

```bash
pip install pygame
```
