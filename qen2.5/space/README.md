# Space Shooter Game

A simple space shooter game created with pygame by qwen2.5-coder:14b on novita.ai.

## Description

A classic space shooter where you control a spaceship, shoot bullets at enemies, and try to survive as long as possible. The game features basic collision detection and enemy spawning.

## Requirements

- Python 3.x
- pygame library

## Installation

First, install pygame:

```bash
pip install pygame
```

## How to Run

```bash
cd qen2.5/space
python main.py
```

## Controls

- **Left Arrow** - Move left
- **Right Arrow** - Move right
- **Spacebar** - Shoot bullets
- Close window or Ctrl+C to exit

## Gameplay

- Control your spaceship at the bottom of the screen
- Enemies fall from the top with varying speeds
- Shoot enemies with bullets to destroy them
- Avoid colliding with enemies
- Game ends when you hit an enemy

## Known Issues

- **Missing pygame dependency**: Make sure pygame is installed with `pip install pygame`
- No score display implemented (font is defined but score tracking is not added)
- Game abruptly ends on collision without showing "Game Over" screen
- No difficulty progression or levels

## Technical Details

- Screen size: 800x600 pixels
- 60 FPS target frame rate
- 8 enemies spawn initially
- Enemies respawn when destroyed or when they leave the screen
